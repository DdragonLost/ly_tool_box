# SQLiteCat

一个用于操作本地 SQLite 数据库的小工具（Python + Tkinter，零第三方依赖）。

## 功能

- 打开本地 `.db/.sqlite/.sqlite3` 文件
- 浏览表/视图列表、查看表结构（字段、主键）
- 分页浏览表数据（LIMIT/OFFSET）
- SQL 控制台：执行任意 SQL，结果表格展示
- 导出当前查询结果为 CSV
- 基础行操作：对选中行按主键更新、删除

## 运行

确保你的 Python 已自带 Tk（Windows 通常默认包含）。

```bash
python app.py
```

## 使用说明

1. 启动后点击“打开数据库”选择文件
2. 左侧选择表
3. 右侧“数据浏览”页可分页查看、更新/删除选中行、导出 CSV
4. 右侧“SQL 控制台”页可输入 SQL 并执行，支持导出结果

## 注意

- 这是本地工具，不做权限/SQL 注入防护；请只打开可信数据库文件。
- 更新/删除依赖表存在主键；若无主键，会提示无法进行该操作。
