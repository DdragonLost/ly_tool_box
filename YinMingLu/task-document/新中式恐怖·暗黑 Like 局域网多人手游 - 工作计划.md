# 新中式恐怖·暗黑 Like 局域网多人手游 - 工作计划

## 项目概述

**项目名称**：新中式恐怖·暗黑 Like 局域网多人手游  
**版本**：v3.0（五代十国·人鬼交织版）  
**目标平台**：iOS / Android  
**游戏类型**：动作 RPG / 暗黑 Like / 局域网多人联机 / 新中式恐怖  
**预计开发周期**：28-38 周（7-9 个月）  
**预算估算**：¥600,000 - ¥1,068,000

---

## 一、项目里程碑

```
2026-Q2 (4-6月): 原型验证 + 可玩 Demo
2026-Q3 (7-9月): MVP 完整版（第一章+第二章）
2026-Q4 (10-12月): 内容扩展（第三章+第四章+第五章）
2027-Q1 (1-3月): 测试优化 + 正式上线
```

---

## 二、详细开发计划

### Phase 1: 原型验证（4 周）

**时间**：2026年4月第1周 - 第4周  
**预算**：¥30,000  
**团队**：1 程序 + 1 策划

| 周次 | 核心任务 | 交付物 | 验收标准 |
|------|----------|--------|----------|
| W1 | 环境搭建 + 基础框架 | Unity 项目框架 | 能创建空场景并运行 |
| W2 | 单机战斗原型 | 战斗 Demo | 角色能移动、攻击、击杀怪物 |
| W3 | 网络联机验证 | 联机 Demo | 2 台设备能联机，看到彼此移动 |
| W4 | 核心循环验证 | 验证报告 | 测试者能理解"打怪→捡装备→变强"循环 |

**关键产出**：
- [ ] Unity 2022.3 LTS 项目框架
- [ ] 基础角色控制器（移动、攻击、闪避）
- [ ] Mirror 网络库集成
- [ ] 简单怪物 AI
- [ ] 核心循环验证报告

**风险评估**：
- 风险：局域网同步不稳定
- 缓解：早期验证，准备备用方案（Photon P2P）

## 四、技术栈

### 引擎与工具

| 类别 | 技术方案 | 理由 |
|------|----------|------|
| 游戏引擎 | Unity 2022.3 LTS | 跨平台、成熟、Asset Store 丰富 |
| 网络方案 | Mirror | 免费、专为局域网设计、文档齐全 |
| 版本控制 | Git + Git LFS | 大文件支持 |
| 项目管理 | 飞书多维表格 | 轻量、协作友好 |
| 设计协作 | Figma | UI 设计、原型 |
| 持续集成 | GitHub Actions | 自动化构建 |

### AI 辅助开发

| 环节 | AI 工具 | 用途 | 节省成本 |
|------|---------|------|----------|
| 代码生成 | Cursor / Copilot | 基础逻辑、网络同步 | 40% |
